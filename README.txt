基幹システムの各種検索・集計

概要
・検索項目を選択、条件を入力し送信すると、結果を一覧で表示するSPA
・DBのMST_QUERYテーブルに実行するSQLの名前と本体を登録しておくことで、任意の集計表を出力できる
・Expressサーバー側でDBとの接続、データの取得を行う
・Expressサーバーには、Reactアプリが動いている開発サーバー：webpack-dev-serverをプロキシサーバーとして使い間接的にアクセス（プロダクション用のbuildは未実行）
・結果の一覧はxlsxファイル形式でダウンロードできる

node.js, express, react(create-react-app)
SQL Server EXPRESS(Tedious)

画面
・検索項目を選択し、それに応じて必要な検索条件入力ボックスが切り替わる
・条件が入力されるまで送信ボタンは無効
・検索結果は100件ずつ表示され、スクロールによって追加分が随時表示される
・exceljsを用いて、検索結果をxlsxファイル形式でダウンロードできる
